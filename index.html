Radium - the PHP micro web application framework -

Radium は、素早いアプリケーション開発 "Rapid Application Development" を行うための軽量なマイクロウェブアプリケーションフレームワークです。

（Work in progress）

動作環境

PHP 5.3+ で動作します。

MVC アーキテクチャ

Radium は、MVC のアーキテクチャを採用しています。

・・・・・

ディレクトリ構造

・・・・・

Model

Model は、MongoDB の O/R マッパーを搭載しています。 MySQL や PostgreSQL 等の一般的な RDBMS を使用することもできます。

・・・・・

View

View は、簡単な PHP ファイルで作成できます。

・・・・・

Controller

Controller は、コントローラクラスの public メソッドを自動的に URL にディスパッチして動作します。
